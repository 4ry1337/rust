> [!INFO] Variable
> A component of a stack frame, either a named function parameter, an anonymous temporary, or a named local variable.

> [!INFO] Local Variable (stack-local allocation)
> Holds a value directly, allocated within the stack’s memory. The value is a part of the stack frame.
